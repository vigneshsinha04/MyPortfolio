Library Database Application

This tool is a desktop based application that can be used in Libraries to store the data of the books present with them. We can see the list of all the books present. We can Add new books, update the details, search the database with custom inputs and even delete the records.

Technical Information:

This desktop application is devloped using Python3. For Graphical User Interface (GUI), I used "Tkinter". For database, I used SQLite3. This application performs basic CRUD operations. First, this tool was a Python Script, later converted to a .exe file.
