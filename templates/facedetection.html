Webcam Face Detector

I have developed a face detection tool that detects straight face of humans in an Image or a Video. If a face is detected, a green rectangle is formed around the face. This is similar to face detection shapes formed when we open our mobile's front camera. Applications of this feature is very wide and one of them is facial recognition software in mobiles and Desktops.

Technical Information:

This tool was developed using opencv for Python. Opencv module is used for image and video processing. The frontal face is detected by using 'Haarcascade'.
