Web Scraping with Python using BeautifulSoup

Web scraping, web harvesting, or web data extraction is data scraping used for extracting data from websites. Web scraping software may access the World Wide Web directly using the Hypertext Transfer Protocol, or through a web browser. While web scraping can be done manually by a software user, the term typically refers to automated processes implemented using a bot or web crawler. It is a form of copying, in which specific data is gathered and copied from the web, typically into a central local database or spreadsheet, for later retrieval or analysis.

For example, we want some set of data from a website for our use. As a programmer, we do not manually go and fetch the data. Instead we let our program do the job. I have developed such tool that fetches data from the famous website called IMDB, particularly the "1012 Movies you must see" section. As we all know that IMDB is a database for movies and TV shows. So, instead of me browsing the entire list of movies, I wanted a list of movies having ratings of more than my selection(for eg. 8). By using this tool, I can get it in seconds. This way, I will save a lot of time and avoid going through the website and filter movies manually. I have developed this tool in Python using BeautifulSoup.

In the selection below, please select an rating and you will get the result of all the movies and their respective IMDB links that have more ratings more than your selection.
